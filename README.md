使い方)  
1)モデルを出力する/モデルを購入する  
2)レンズを用意する  
3)モデル表面に穴が一か所空いているモデル(一部のBooth販売版は穴ではなく黒い目印がつけられています）が右、空いていないのが左   
4)出力したアダプターにレンズを差し込んでみる  
5)enjoy
