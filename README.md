# VR-Lens_for_WindowsMR
VR Lens for WindowsMR


とりあえずモデルだけ公開。後で解説書く


使い方)  
1)モデルを出力する  
2)レンズを用意する  
3)出力したアダプターにレンズを差し込んでみる  
4)作り方によっては若干個体差が出るので、もしぶかぶかだったら接着剤、ボンド等で固定する  
5)enjoy  
  
注意)
接着剤でレンズを固定するのは良い方法です。ただし、接着剤をレンズのふち以外につけて汚さないように十分注意すること。
